# Requirements #

  * Ruby
  * [Rake](http://rake.rubyforge.org/)

# Building #

Run `rake` in the project's root folder.

# Outputs #

  * user.js scripts for The Movide Dude and The Game Dude in the _fill this in_ folder.
  * A tarball of the entire project in the `dist` folder.
